ABSTRACT

The subject of this development project is a website for foreign language practice. It is intended to serve as an informational and educational resource that presents the platform services and supports user interaction with learning content.

The work includes an analysis of the project domain, formulation of requirements for the user interface and functionality, and selection of the most appropriate web technologies for implementation.

The website is developed using HTML5, CSS3, and JavaScript. In the current version of the project, data storage is implemented with the client-side NoSQL mechanism IndexedDB. A separate service layer, data-service.js, is used to organize unified access to posts, requests, lessons, and authorization state.

As a result of this work, an adaptive multi-page website has been developed. It includes a homepage, services section, lessons section, blog, contacts page, publication creation page, and request management page. The current implementation satisfies the major requirements of the project.
