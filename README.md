# Lingoflux: сайт для языковой практики

Демонстрационное клиент-серверное веб-приложение для просмотра учебных материалов, публикаций и отправки заявок. Для сотрудника предусмотрены авторизация и управление уроками, публикациями и обращениями пользователей.

## Функции

- просмотр главной страницы, услуг, уроков и блога;
- открытие полной публикации;
- отправка заявки через форму обратной связи;
- создание и удаление уроков и публикаций после входа;
- просмотр и удаление заявок пользователей;
- хранение данных в `backend-data.json`.

## Запуск

```powershell
node server.js
```

После запуска сайт доступен по адресу:

```text
http://localhost:8000
```

Также можно использовать PowerShell-скрипт:

```powershell
.\serve.ps1
```

## Основные файлы

- `index.html`, `services.html`, `lessons.html`, `blog.html`, `contacts.html` — пользовательские страницы;
- `login.html`, `create-post.html`, `requests.html` — служебные страницы;
- `styles.css` — оформление и адаптивная верстка;
- `main.js` — общая логика интерфейса;
- `data-service.js` — единый слой обращения к данным и API;
- `server.js` — backend на Node.js;
- `backend-data.json` — демонстрационное JSON-хранилище.

## C++ backend showcase

В репозитории также есть отдельный демонстрационный backend-проект на C++20:

```text
cpp-backend-showcase/
```

Он показывает навыки backend-разработки на C++: HTTP-обработку, роутинг, bearer-token авторизацию, потокобезопасное хранилище, JSON, метрики, CMake-сборку и unit-тесты. Подробности и команды сборки находятся в [`cpp-backend-showcase/README.md`](cpp-backend-showcase/README.md).

## Ограничения

Это демонстрационная версия. Для промышленной эксплуатации нужно добавить хеширование паролей, HTTPS, CSRF-защиту, ограничение частоты запросов, постоянное хранилище сессий и полноценную СУБД.
